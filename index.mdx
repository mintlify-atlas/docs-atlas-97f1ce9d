Whether you're reading papers, annotating PDFs, or building a knowledge base, Better Notes connects all your ideas in one place — without leaving Zotero.

## Get started in minutes

<Steps>
  <Step title="Install Better Notes">
    Download the `.xpi` file from the [latest release](https://github.com/windingwind/zotero-better-notes/releases/latest) and install it through **Tools → Plugins** in Zotero.
  </Step>
  <Step title="Open a note tab">
    Double-click any note in your Zotero library to open it in a full tab with the outline panel, editor, and context pane.
  </Step>
  <Step title="Create note links">
    Click the Better Notes icon in the editor title bar to link related notes together, building a connected knowledge graph.
  </Step>
  <Step title="Sync with Markdown">
    Click **Set Auto-Sync** when exporting a note to keep it automatically synced with an external Markdown file for use with Obsidian or any Markdown editor.
  </Step>
</Steps>

<Note>
  Better Notes targets Zotero 8 and 9. Make sure you are running a compatible version before installing.
</Note>
