---
title: "Get started with Better Notes in 5 minutes"
description: "A hands-on walkthrough of the core Better Notes workflow: opening a note tab, using the Magic Key, creating note links, and setting up Markdown sync."
---

Better Notes offers a range of features that can be combined like building blocks to create your own note-taking workflow. This guide walks you through the most important ones — from opening your first note tab to syncing with an external Markdown file — in about 5 minutes.

<Steps>
  <Step title="Open a note in a tab">
    In your Zotero library, double-click any note item (or select it and press `Enter`) to open it in a full-width note tab — the same way you open a PDF attachment.

    To open the note in a separate window instead, hold `Shift` while double-clicking.

    <Tip>
      In earlier versions of Better Notes, the main note view was called the "workspace." That concept no longer exists — you can now open as many note tabs and windows as you like, all at the same time.
    </Tip>
  </Step>
  <Step title="Explore the note tab layout">
    Once a note is open in a tab, you will see three panels arranged left to right:

    - **Outline** — a live table of contents generated from your headings, useful for navigating long notes
    - **Note editor** — the main writing area, powered by ProseMirror with Better Notes enhancements
    - **Context pane** — shows tags, related items, a visual relation graph, and inbound/outbound note links

    <Note>
      You can also open the note editor in two other places: click a note in the library to show a compact editor in the **item pane**, or open a PDF and find the note editor in the **right-side context pane** of the PDF reader.
    </Note>
  </Step>
  <Step title="Create your first note">
    To create a new standalone note:

    - Click the **note icon** in the library toolbar (the row of icons directly below the tab bar), or
    - Go to **File → New Note**

    To create a note attached to a library item, right-click the item and choose **Add Note**.
  </Step>
  <Step title="Use the Magic Key">
    Better Notes adds a **Magic Key** command palette to the note editor. Open it by:

    - Typing `/` on a new line in the editor, or
    - Pressing `Ctrl`+`/` (Windows/Linux) or `Cmd`+`/` (macOS)

    Use the arrow keys to browse available commands — insert templates, add headings, create links, and more — then press `Enter` to execute.

    <Tip>
      The Magic Key is the fastest way to insert a note template or apply formatting without leaving the keyboard.
    </Tip>
  </Step>
  <Step title="Create a note link">
    Note links connect related notes together and power the relation graph in the context pane.

    To link the note you are currently editing to another note:

    1. Click the **Better Notes icon** (the BN logo) in the title bar of the note editor.
    2. A link creator dialog opens. Choose **Mention in** or **Link to**:
       - **Mention in** inserts a link to the current note inside another note you pick — this creates an inbound link.
       - **Link to** inserts links from one or more picked notes into the current note — these are outbound links.
    3. Select the target note(s) and confirm.

    You can also copy a note's link manually from the note editor context menu and paste it anywhere.

    <Note>
      After creating a link, open the **Context pane** to see the relation graph update in real time.
    </Note>
  </Step>
  <Step title="Set up Markdown sync">
    Better Notes can keep any note automatically in sync with an external Markdown file — useful for editing in Obsidian, VS Code, or any other Markdown editor.

    To enable auto-sync for a note:

    1. With the note open, click the **export** button in the editor toolbar.
    2. Choose **Markdown** as the export format and pick a save location.
    3. When prompted, click **Set Auto-Sync**.

    From this point on, any changes you make in Zotero will be written to the Markdown file, and any edits to the file will be reflected back in Zotero after the auto-sync period elapses (configurable in preferences).

    <Tip>
      Sync runs after you close the note editor and after the sync period you set in **Tools → Better Notes Preferences**. You do not need any third-party sync tool.
    </Tip>
  </Step>
</Steps>
