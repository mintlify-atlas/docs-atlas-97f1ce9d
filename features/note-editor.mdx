title: "Note editor enhancements in Better Notes for Zotero"
description: "Explore the note editor features that Better Notes adds to Zotero: Magic Key palette, link preview, Markdown paste, outline panel, and more."
---

Better Notes enhances Zotero's built-in note editor with a command palette, link previews, direct Markdown paste, an outline panel, a context pane with relation graphs, and image management tools. These features are available wherever the note editor appears in Zotero.

## Where to find the note editor

Better Notes enhancements apply to the note editor in all three locations where Zotero shows it:

- **Note tab or window** — double-click a note item in your library, or hold `Shift` while pressing `Enter` to open it in a new window. This is the full layout with an outline on the left and a context pane on the right.
- **Item pane** — click a note in the library to open a note editor in the right-side item pane.
- **PDF reader context pane** — the right-side pane of the PDF reader shows a note editor for the PDF's linked note.

<Tip>You can open as many note tabs and windows as you like at the same time. There is no single "workspace" window anymore.</Tip>

To create a new note, click the note icon in the library toolbar (the row beneath the tab bar), or go to **File** in the menu bar.

## Magic Key command palette

The Magic Key is a command palette built into the note editor. Open it by typing `/` anywhere in the editor body, or by pressing `Ctrl`+`/` (Windows/Linux) or `Cmd`+`/` (macOS).

Once open, you can:

- **Type** to filter available commands
- **Arrow keys** to navigate the list
- **Enter** to execute the selected command

The palette provides quick access to formatting options, template insertion, and other editor actions without leaving the keyboard.

## Note link preview

When you hover over a note link while holding `Ctrl` (Windows/Linux) or `Cmd` (macOS), Better Notes shows a preview of the linked note inline. You can also click the link to open the preview pane, letting you read linked content without navigating away from the note you are editing.

## Direct Markdown paste

You can paste Markdown-formatted text directly into the note editor. Better Notes automatically converts the pasted Markdown to Zotero's rich text format. This means content copied from Markdown editors, terminals, or web pages retains its structure when pasted.

## Outline panel

When a note is open as a tab, a document outline appears on the left side of the note tab. The outline reflects the heading structure of your note and lets you jump to any section with a single click.

<Note>The outline panel is only visible in the note tab layout. It is not shown in the item pane or PDF reader context pane.</Note>

## Context pane

The right side of a note tab contains a context pane with several panels:

<CardGroup cols={2}>
  <Card title="Tags" icon="tag">
    View and manage tags attached to the note.
  </Card>
  <Card title="Related" icon="link">
    See items that Zotero has marked as related to this note.
  </Card>
  <Card title="Relation graph" icon="diagram-project">
    A visual graph showing the note's connections to other notes via note links.
  </Card>
  <Card title="In/outbound links" icon="arrow-right-arrow-left">
    Lists all notes that link to this note (inbound) and all notes this note links to (outbound).
  </Card>
</CardGroup>

## Image management

Better Notes adds two image actions to the note editor:

- **Resize** — right-click an image to open a resize prompt where you enter a new pixel width.
- **Preview** — double-click or `Ctrl`/`Cmd`+click an image to open a full-size viewer showing all images in the note.

These actions are also accessible via the image popup toolbar that appears when you select an image in the editor.

## Quick Note from annotation

In the PDF reader, you can convert any annotation to a standalone note with one click using the **Quick Note** button. Better Notes uses the `[QuickNoteV5]` built-in template to generate the note content from the annotation, including the annotation comment and highlighted text.

<Tip>You can customize how Quick Notes are generated by editing the `[QuickNoteV5]` template in the Note Template Editor. See [Note templates](/features/note-templates) for details.</Tip>
