Better Notes is distributed as a single `.xpi` file that you install directly through Zotero's plugin manager. No external tools, no command line — just download and install.

## System requirements

- **Zotero 8 or 9** — Better Notes does not support older versions of Zotero.

<Warning>
  Beta releases of Better Notes are built against Zotero beta. Do not install a beta `.xpi` on a stable Zotero release, or an unstable `.xpi` on Zotero beta — only install a release that matches your Zotero channel.
</Warning>

## Download the plugin

Download the `.xpi` file from the GitHub releases page:

- [Latest stable release](https://github.com/windingwind/zotero-better-notes/releases/latest)
- [All releases](https://github.com/windingwind/zotero-better-notes/releases) — includes beta and older versions

<Note>
  If you are using Firefox as your browser, right-click the `.xpi` link and select **Save Link As…** (or **Save As…**). Firefox may otherwise try to install the file as a browser extension rather than downloading it.
</Note>

## Install the plugin

<Steps>
  <Step title="Open the Plugins manager in Zotero">
    In Zotero, click **Tools** in the top menu bar, then click **Plugins**.
  </Step>
  <Step title="Open the gear menu">
    On the Extensions page that opens, click the **gear icon** (⚙) in the top-right corner.
  </Step>
  <Step title="Select Install Add-on From File">
    Click **Install Add-on From File…** in the dropdown menu.
  </Step>
  <Step title="Select the downloaded .xpi file">
    Browse to the location where you saved the `.xpi` file, select it, and click **Open**.
  </Step>
  <Step title="Restart if prompted">
    Zotero may ask you to restart to finish the installation. Click **Restart Now** if prompted.
  </Step>
</Steps>

After restarting, you will see the Better Notes icon appear in the note editor title bar, and the **Tools** menu will include new Better Notes entries such as **New Template from Clipboard** and **Note Template Editor**.
